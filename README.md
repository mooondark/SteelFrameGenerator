🇫🇷 [🇬🇧](README_EN.md)

Générateur de portiques simples avec l'API d'Advance Design

## Fonctionnalités

- Génération de portiques 3D
- Créations de parois
- Génération du poids propre
- Langues incluses : FR/EN/PL
- Sauvegarde des options
